# JCAS_Multibeam_Optimization
#Tối ưu hóa JCAS Multibeam dùng PSO

Sinh viên thực hiện: Lương Minh Tuấn
Mã sinh viên: 20215158

## Mô tả
Dự án này thực hiện tối ưu hóa búp sóng đa chùm tia (Multibeam) cho hệ thống JCAS sử dụng các biến thể của thuật toán PSO:
- Classic PSO
- LIDW-PSO
- X-PSO
- N-PSO

## Hướng dẫn chạy
1. Tải toàn bộ code về máy.
2. Mở MATLAB.
3. Chạy file `main.m`.
